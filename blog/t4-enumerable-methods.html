Further research of the #map enumerable!

May 29, 2015

An enumberable is a "module", which is fancy for a "a bunch of methods". Enumberables can be used to sum up methods and shorten code as long as the class using the enumberable has its own #each method, such as a hash or array.

#map is handy the enumberable I will be focusing on in this blog. #map is very similar to #collect. It will interate your block on every item in your object, whether it be an array or hash. It will then keep each item but change it and then return the results. But, it will not modify the original object unless you make it destructive.

Below is a simple example that helped solidify my understanding:
[1,2,3].map {|x| x + 1 }
#=> [2,3,4]
