You're about to take your assessment. This can be a really scary thing, but you won't let it be. Remember not to panic, you have the tools you need to get through this. Read the errors and breathe! Take a look at the resources below to get you out of jams!

variables:
name = "Davina"
>> name
=> "Davina"

names = ["Davina", "Lucas"]
>> names[1]
=> "Lucas"
>> names[0]
=> "Davina"

hash:
names = {student: "Davina", teacher: "Lucas"}
>> names[:teacher]
=> "Lucas"
>> names[:student]
=> "Davina"

object:
person = Person.new({name: "Davina"})
>> person.name
=> "Davina"
